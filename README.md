# una-ihcux-lista-11
Letícia

O beneficie é evitar o descarte indevido de certos itens.

Na tela de login o usuário faz seu cadastro, na busca vc pode procurar seu item, no chat vc pode conversar com o doador, no seu perfil vc pode olhar seus selos Eco, e no cadastro de item vc pode colocar o item que vc quer doar e as informações dele.

Caminho receber: Faz o login, procura produto, ver detalhes do produto, escolhe o produto, conversa com o doador, pega seu item, com o tempo pode conseguir selos no perfil. 

Caminho doador: Faz o login, cadastra itens, esperar pessoa interressada, conversar com a pessoa, enviar o item, com o tempo tambem pode conseguir selos no perfil.
